-MVVM </br>
-Navigation </br>
-ViewModel&LiveData </br>
-Retrofit&RxJava </br>
-Refresh Layout </br>
-Room </br>
-Coroutine </br>
-DI(Hilt) </br>
-Repository </br>
-Testing(Hilt,RecyclerView,Room,Fragment) </br>
